Duolingo Vocab
=============

A Chrome extension to assist in learning vocabulary on duolingo.com

## Current Features
+ Query Duolingo to get list of known vocab
+ Reformat hints and word data from Duolingo
+ Quiz current list of vocab
+ Angular
